# 羽毛球课时记录系统

一个简单的羽毛球教学管理系统，用于记录学员课时、费用统计和数据分析。

## 功能特点

- 🏸 记录学员课时信息
- 💰 费用统计管理
- 📊 数据可视化图表
- 📱 响应式设计，支持手机访问
- 💾 本地存储，无需服务器

## 技术栈

- HTML5
- CSS3
- JavaScript
- Chart.js 图表库

## 使用方法

1. 打开网页
2. 添加学员课时记录
3. 查看统计图表
4. 管理历史记录

## 文件结构

- `index.html` - 主页面
- `style.css` - 样式文件
- `script.js` - JavaScript 逻辑
